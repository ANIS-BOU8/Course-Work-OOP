# Система продажи авиабилетов

## Описание

Этот проект реализует систему для управления и продажи авиабилетов с использованием принципов объектно-ориентированного программирования (ООП). Система позволяет создавать рейсы, продавать билеты, изменять данные рейсов и билетов, а также просматривать проданные билеты.

## Возможности
- Создание новых рейсов
- Бронирование билетов (выбор класса и места)
- Изменение данных о рейсах и билетах
- Просмотр проданных билетов

## Технологии
- Java
- Объектно-ориентированное программирование (ООП)
